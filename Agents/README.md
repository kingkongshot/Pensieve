# Agent 配置使用说明

## 📋 快速开始

`AGENTS.md` 是一个 AI Agent 角色配置模板，定义了 AI 助手的行为方式和工作流程。如果你使用 Cursor 或者 Windsurf、Augment 等 AI 开发工具，那么放置在仓库的根目录即可。如果使用 Claude Code，则可以重命名为 CLAUDE.md 放在仓库根目录。

## 🔧 需要修改的关键信息

使用此模板前，请根据你的实际情况修改以下内容：

### 1. **项目名称** (第3行)
```markdown
现在我们正在开创一个 "你的项目" 的新项目
```
**修改为：** 你的实际项目名称和描述

---

### 2. **我的人格类型** (第7行)
```markdown
我是 INFP 人格，常见陷阱有：
```
**修改为：** 你自己的 MBTI 人格类型及其常见陷阱

**参考：** 可查看项目中的 [MBTI_DEV_TRAPS.md](./MBTI_DEV_TRAPS.md) 找到自己的 MBTI 对应人格的特点复制进去。

---


### 5. **工具使用** (第166-168行)
```markdown
## 工具使用
这里描述 AI 可以使用的各类工具。例如：
- `get_code_context_exa` - ...
```
**修改为：** 列出你的 AI 助手可以使用的实际工具和 API

---

### 6. **仓库指南** (第171-172行)
```markdown
## 仓库指南
这里描述项目的文件结构，以及基础命令、常用、代码风格等。
```
**补充：** 你的项目具体信息
- 文件结构说明
- 常用命令（如 `npm run dev`、`pytest` 等）
- 代码风格规范
- Git 工作流程
